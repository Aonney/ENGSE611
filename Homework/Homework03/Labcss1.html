ตัวอย่างการ์ดด้วย Flexbox

Card 1

AI (Artificial Intelligence) หรือ ปัญญาประดิษฐ์ คือ เทคโนโลยีที่ทำให้คอมพิวเตอร์และเครื่องจักรสามารถเลียนแบบความสามารถทางปัญญาของมนุษย์ เช่น การเรียนรู้ การตัดสินใจ การแก้ปัญหา และการสื่อสาร โดยอาศัยการประมวลผลข้อมูลจำนวนมหาศาลผ่านอัลกอริทึม ทำให้สามารถทำงานที่ซับซ้อนได้เองโดยอัตโนมัติ และสามารถพัฒนาความสามารถได้ต่อเนื่องจากข้อมูลใหม่ๆ

Card 2

HTML (HyperText Markup Language) คือภาษาหลักที่ใช้สร้างโครงสร้างพื้นฐานของหน้าเว็บ โดยใช้ "แท็ก" (Tags) ในเครื่องหมาย < > เพื่อจัดระเบียบเนื้อหา เช่น กำหนดหัวข้อ, ย่อหน้า, รูปภาพ และลิงก์ เพื่อบอกเว็บเบราว์เซอร์ว่าควรแสดงผลอย่างไร เปรียบเสมือนโครงกระดูกของเว็บไซต์ที่ทำงานร่วมกับ CSS (ตกแต่งหน้าตา) และ JavaScript (เพิ่มลูกเล่น) เพื่อสร้างเว็บไซต์ที่สมบูรณ์

Card 3

CSS (Cascading Style Sheets) คือภาษาที่ใช้กำหนด รูปลักษณ์และการจัดวาง ของหน้าเว็บ (HTML) เช่น สี ตัวอักษร ขนาด ระยะห่าง และเค้าโครง เพื่อให้เว็บดูสวยงามและน่าสนใจ โดยแยกหน้าที่จาก HTML ที่ทำหน้าที่สร้างโครงสร้างเนื้อหา ทำให้พัฒนาและดูแลรักษาง่ายขึ้น มีประสิทธิภาพในการสร้างเว็บที่ตอบสนอง (Responsive) และสม่ำเสมอทั่วทั้งเว็บไซต์.
